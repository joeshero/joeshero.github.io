对于对象demo1，克隆一个对象demo2，并打印demo2，当再次修改demo1的值，demo2的值也会变，似乎二者共用了某些变量。

实际上，对于基本变量，demo2直接复制了一份来用，所以修改demo1的基本变量不会影响demo2的值。而对于引用变量，二者共用同一个引用变量。对于clone来说，这是一个浅拷贝。
